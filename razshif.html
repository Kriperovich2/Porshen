ПОРШЕНЬ

Система использует только буквы «П О Р Ш Е Н Ь» в качестве символов шифра.
Алфавит шифра «ПОРШЕНЬ»
А → ПП Б → ПО В → ПР Г → ПШ Д → ПЕ Е → ПН Ё → ПЬ Ж → ОП З → ОО И → ОР Й → ОШ К → ОЕ Л → ОН М → ОЬ Н → РП О → РО П → РР Р → РШ С → РЕ Т → РН У → РЬ Ф → ШП Х → ШО Ц → ШР Ч → ШШ Ш → ШЕ Щ → ШН Ъ → ШЬ Ы → ЕП Ь → ЕО Э → ЕР Ю → ЕШ Я → ЕЕ Пробел → ЕН Точка → ЕЬ Запятая → НП Цифры 0–9 → НО, НР, НШ, НЕ, НН, НЬ, ЬП, ЬО, ЬР, ЬШ
Совместимо с кириллическими буквами (А–Я, включая Ё). Нажмите «Наоборот», чтобы зашифровать; «Расшифровать» — чтобы получить исходный текст.
